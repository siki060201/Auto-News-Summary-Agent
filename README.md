# 🤖 Auto-News-Summary-Agent (自动化资讯摘要助手)

## 📌 项目简介
本项目是一个基于大语言模型构建的轻量级自动化 Agent。旨在解决信息过载问题，自动抓取指定 RSS 订阅源或网页内容，通过 LLM 进行长链推理与信息清洗，最终生成结构化的核心摘要报告。

## 🎯 解决的核心痛点
日常开发和运营人员需要花费大量时间浏览各个科技网站（如 HackerNews、知乎等）获取行业动态。人工阅读耗时且容易遗漏重点，本项目通过多 Agent 协作将该流程完全自动化。

## ⚙️ 核心逻辑流
1. **数据抓取模块 (Scraper)**: 定时读取目标站点的最新文章列表。
2. **多 Agent 协作推理**:
   - **分发 Agent**: 判断文章类别（AI、前端、后端等）。
   - **总结 Agent**: 提取文章的核心论点、数据和结论，过滤废话。
3. **输出模块 (Formatter)**: 将总结内容整合为 Markdown 格式，推送到飞书/钉钉机器人或保存为本地文档。
